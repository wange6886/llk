# 流量卡举报助手 🛡️

**3 分钟生成投诉方案，一键直达举报入口**

被 B 站 / 抖音 / 小红书上的虚假流量卡广告骗了？激活后发现套餐严重缩水、隐形扣费、注销困难？这个工具帮你高效维权。

## 功能

| 功能 | 说明 |
|------|------|
| 📝 智能表单 | 输入手机号自动识别运营商，填写基本信息 |
| 🎯 场景识别 | 自动识别 5 大投诉场景（虚假宣传 / 限速 / 扣费 / 注销难 / 流量虚标） |
| 📄 话术生成 | 每个场景提供 3 种话术（简短版→平台举报、标准版→客服投诉、完整版→工信部申诉） |
| ⚖️ 法律引用 | 话术自动引用《广告法》《消费者权益保护法》相关条款 |
| 🔗 直达入口 | 平台举报链接 → 运营商客服热线 → 工信部 / 12315，按优先级排列 |
| 📋 证据清单 | 按场景生成证据收集清单，支持进度追踪和 .txt 下载 |
| 💾 历史记录 | 投诉记录本地保存，支持查看、恢复和删除 |

## 支持平台

- **推广平台**：抖音 · 小红书 · B站
- **运营商**：移动 · 联通 · 电信 · 广电
- **申诉渠道**：平台举报中心 · 运营商客服 · 工信部申诉 · 12315

## 使用方式

纯静态页面，无需安装，浏览器直接打开即可使用。

### 在线访问

GitHub Pages 部署后即可访问。

### 本地使用

```bash
# 直接用浏览器打开
open index.html
```

或者用任意 HTTP 服务器托管：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

然后在浏览器打开对应地址即可。

## 技术栈

- 纯 HTML + CSS + JavaScript，零依赖
- Tailwind CSS（CDN）
- 无构建步骤，无框架
- 兼容微信内置浏览器
- localStorage 本地存储
